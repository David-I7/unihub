# Notification subscriptions combine scope, event type, and schedule

UniHub stores app-generated events as a global activity feed. The first backend version does not include user-specific in-app notification inbox items, read state, or recipient rows. Email delivery is controlled separately by notification subscriptions, where each subscription combines a user, a notification scope, an event type, and a delivery schedule. Notification scopes are limited to series and course; series subscriptions support exam reminders, deadline reminders, lecture reminders, course additions, material changes, and material additions, while course subscriptions support all of those except course additions. Course additions, material changes, and material additions can only use immediate delivery; exam reminders, deadline reminders, and lecture reminders can only use one day before, one week before, or one month before.
